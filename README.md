# STUDIOKOMM - Creative Agency Website

This is the official website for STUDIOKOMM, a creative agency specializing in photography, video production, events, and wedding services based in Marrakech, Morocco.

The website is a single-page application (SPA) built primarily with HTML and Tailwind CSS (using the `@tailwindcss/browser` script for on-the-fly compilation).
